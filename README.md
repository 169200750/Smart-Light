# Smart-Light
基于STM32的智能台灯

### 项目概述
该智能台灯能够实时监测环境光照强度，并根据环境亮度调整LED灯的亮度，亦可通过面板对台灯进行手动控制，从而实现智能护眼和节能的目的。

### 功能描述
1.环境感知功能：实时监测环境光照强度  
2.信息显示功能：显示亮度等提示信息  
3.面板控制功能：模式和光照等级切换  
4.远程控制功能：远程开关LED灯  

### 中断级调度任务
1.面板控制任务  
2.环境感知任务  
3.远程控制任务
